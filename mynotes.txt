لتغيير اللغة للتمبلت 
----------------------
اذهب إلى مجلد موقعك
ثم 
themes
ثم
التمبلت الذي تريد وفي حالتنا هذه هو التمبلت المسمى
PaperMod
ثم
layouts
ثم افتح ملف
baseof.html
وغير السطر الثاني الذي يتحدث عن الدايركشن من 
auto
إلى 
rtl
------------
لاعداد موقع جديد
تحتاج لتركيب 
python
وتحتاج لتركيب
brew
وتحتاج لتركيب 
hugo
من الترمنال اكتب
brew install hugo

ثم انشأ مجلد موقع باتعباع تعليمات 
hugo 
كالتالي
hugo new site quickstart
هذا الأمر ينشأ لك مجلد لموقعك اسمه كويكستارت
ثم
cd quickstart
ثم
git init
الذي يحول المجلد الى مجلد جيت
ثم انسخ ثيم مجلد يعجبك من ثيمات هوجو وانشأ له مجلد باسمه داخل مجلد 
themes
وحط محتويات الثيم كاملة داخل المجلد الجديد
او تقدر لو تبي الديفولت ثيم اللي اسمه انانكي ان تنسخه بالأمر التالي
git submodule add https://github.com/theNewDynamic/gohugo-theme-ananke themes/ananke



ثم تحتاج لحساب جيت هوب
وتحتاج لانشاء 
public repository
ثم اتبع التعليمات لربط مجلد لديك في جهازك بالريبوسيتوري
المجلد يجب أن تدخل عليه من الترمنال وتعمل له الامر التالي
git init

طبعا اذا كنت اول مرة تستخدم جيت تحتاج لانشاء يوزر وبريد عن طريق امرين في 
الترمنال موجودين في النت


---------
التعامل مع أي تمبلت
اضف أي تمبلت يعجبك الى مجلد 
themes
ثم عدل ملف 
config.yml
السطر الذي فيه
theme=
حط بعد اليساوي اسم مجلد التمبلت
----------
